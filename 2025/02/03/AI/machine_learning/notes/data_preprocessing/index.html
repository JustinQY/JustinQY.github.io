Data Preprocessing

Methods of data preprocessing, such as data augmentation when we have no enough data; data cleaning when there is any noise, outliers or missing value in our data.

EDA: Exploratory Data Analysis (探索性数据分析)

Nan Values Processing

There may be some missing values in the training/test set. We should find and handle them properly.

Detect Missing Values

Fill by SimpleImputer

For these features, use different methods to fill the missing values

One-Hot Encoding

For the features contain limited numbers of string classes(e.g. gender: female and male, embarked class: S, C and Q), use One-Hot Encoding to encode.

Outliers Detection

Detect and remove lines which contain outliers using IQR.

Feature Selection (特征选择)

Heatmap + PCA

  • Use heatmap to check correlations between all features.
  • If there are some features that have high correlation(say > 0.8) with each other (this is called: Multi-collinearity(多重共线性)), but all have low correlation(say < 0.2) to target. This means they are redundant features.
  • Use PCA to decrease dimension and remove redundant features.

Normalization and Standardization

Normalization (归一化)

When data is not normal distribution or there is large gap between ranges of different features (e.g. 0~100 and 100~10000).
For feature $X$,
$
X_{normalized} = \frac{X\ -\ X_{min}}{X_{max} - X_{min}}
$

Use in neural networks, KNN, K-means
Scale feature to a fixed range: [0, 1] or [-1, -1].

Standardization (标准化)

When data is normal distribution, scale data to distribute as a standard normal distribution with mean = 0 and standard deviation = 1.
For feature $X$,
$
X_{standardized} = \frac{X\ -\ \mu}{\sigma},\ where\ \mu\ is\ mean\ of\ X,\ \sigma\ is\ standard\ deviation\ of\ X
$

Some models or algorithms (linear/logistic regression, SVM, PCA) are sensitive to data distribution, use Standardization to make standard normal distribution.

In binary/multi classification problems, there may be an imbalance data size between categories, like healthy people/patients, normal/fraud trades.

SMOTE (Synthetic over-sampling 合成过采样)
